# Huffways commpressing tool
 
Taking text files as input and output a compressed/crypted  file(with .hfw extention) and a decompession table (table.hft ).

### Prerequisites
 
Any machine that run c++ in it (has g++ already intalled )
## Built With

* C++
* Codeblocks
* OOP
